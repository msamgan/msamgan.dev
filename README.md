# Base Template

This is a base template for a laravel project.

## Features

- [x] Laravel 11
- [x] PHP 8.*
- [x] Role Based Access Control
- [x] User Management
- [x] User Profile

## To Make a new Module

```bash
php artisan app:make:module
```
This command will create the following files:
- Controller
- Model
- Migration
- Factory
- Seeder
- Request
- Routes
- Views
- Permissions
- Menu
